# 3-Statement Financial Model (Excel)

This repository contains a fully integrated three-statement financial model built in Microsoft Excel.

## Overview
- Integrated Income Statement, Balance Sheet, and Cash Flow Statement
- Multi-year financial projections
- Fixed asset schedule with depreciation linkage
- Debt schedule including interest expense and repayments
- Retained earnings linked through net income
- Balance Sheet balanced across all forecast periods

## Purpose
This project was created to strengthen practical understanding of financial statement interlinkages, accounting logic, and core corporate finance concepts.

## Tools Used
- Microsoft Excel
